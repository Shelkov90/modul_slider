Опции

Список, включающий все варианты настраиваемых опций

containerId

Тип: String
Default: 'slider-container'

Название ID контейнера со слайдером

slideClass

Type: String
Default: 'slide'

Название класса div'a с фото

title

Type: String
Default: 'Slider'

Название заголовка слайдера

prevArrow

Type: String
Default: 'prevArrow'

Название ID стрелки предыдущего слайда

nextArrow

Type: String
Default: 'nextArrow'

Название ID стрелки следующего слайда

nav

Type: String
Default: 'false'


Допустимые значения: 'true'/'false'/

Навигация слайдера

autoplay

Type: String
Default: 'false'


Допустимые значения: 'true'/'false'/

Автовоспроизведение слайдов
