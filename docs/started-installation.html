Установка

Подключение CSS

Сначала включите файл CSS в HTML head:

<link rel="stylesheet" href="slider.css">

Подключение JS

Подключите скрипты в указанном порядке перед закрывающим тегом body

              
                <script src="slider.js"></script>
                <script src="script.js"></script>

Структура HTML

Вы можете использовать эту структуру HTML или создать свою.Главное придерживаться названий классов и id. Вы также можете использовать другой класс или id. В таком случае при подключении плагина необходимо указать соответствующий класс. 

              
                <!-- Set up your HTML -->

                <div id="slider-container" class="slider-container">
                <h2 id="slider-title">My Slider</h2>
                <div class="slider">
                    <div class="slide active">
                        <img src="img/7.avif" alt="Slide 1">
                    </div>
                    <div class="slide">
                        <img src="img/8.avif" alt="Slide 2">
                    </div>
                    <div class="slide">
                        <img src="img/9.avif" alt="Slide 3">
                    </div>
                </div>
                <div class="prev-arrow" id="prevArrow">←</div>
                <div class="next-arrow" id="nextArrow">→</div>
            </div>

Вызов плагина

Теперь вызовите функцию инициализации и ваш слайдер готов.

let mySlider3 = new Slider({ 
              containerId: 'slider-container3',
              slideClass: 'slide',
              title: 'My Slider',
              prevArrowId: 'prevArrow',
              nextArrowId: 'nextArrow',
              nav: 'true',
              autoplay: 'false',
            });
